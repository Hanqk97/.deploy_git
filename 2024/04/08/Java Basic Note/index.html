Data Type

  • Integer Types
    • byte: Occupies 1 byte of computer storage, representing a range from -128 to 127.
    • short: Takes up 2 bytes of storage, with a range from -32,768 to 32,767.
    • int: 4 bytes, with a range of approximately -2^31 to 2^31-1.
    • long: 8 bytes, with a range of approximately -2^63 to 2^63-1.
  • Floating Point Types
    • float: Single precision floating point, occupies 4 bytes, often used to store smaller floating point numbers.
    • double: Double precision floating point, takes up 8 bytes,typically used for storing larger or more precise floating point numbers.
  • Character Type
    • Used to store a single character (enclosed in single quotes), such as ‘A’ and ‘1’.
  • Boolean Type
    • Represents a boolean value, only two possible values: true and false.

Relational Operator

  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to
  • !=: Not equal to

Logical Operator

  • Logical AND operator &&: The entire result is true only if both conditions on the sides of the operator are true.
  • Logical OR operator ||: The result is true if at least one of the conditions on the sides of the operator is true.
  • Logical NOT operator !: This operator returns the opposite of the condition’s value, turning true to false and false to true.

If Condition

  • break: break used to terminate the nearest enclosing while, do while, or for loop. No code following the break within the loop will execute after it is called.
  • continue: continue appear inside for, while, and do while loops. It is used to skip the remaining part of the current loop iteration and then proceed with the next iteration.

Array

  • Fixed Size: Once an array is declared, its size is usually fixed and cannot be dynamically changed during runtime.

  • Same Data Type: All elements in the array must be of the same data type.

  • Continuous Storage: Elements in an array are stored continuously in memory.

  • Index Access: Elements in an array are accessed through an index (or subscript).

  • Dynamic Initialization: After declaring an array, use the new keyword to allocate memory and initialize the elements.

  • Static Initialization: When declaring an array, memory is allocated, and initial values are specified simultaneously.

  • Length: All arrays possess a length attribute, which is used to obtain the length of the array, indicating the number of elements it contains.
